### Authentication

You need an API key. You can obtain your API key by signing up for an account on our platform. Use this API key in the Authorization header of your HTTP requests.



<Callout type="warning" emoji="⚠️">
 Make sure to never post your API publicly
</Callout>

Access token programmatically authenticate your identity to the developer api hub,

[Visit the documentation to know more](https://hushh.gitbook.io/hushh-docs)
